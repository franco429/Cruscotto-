CRUSCOTTO LOCAL OPENER
======================

Questo software è un'applicazione sicura e certificata per aprire documenti 
direttamente dal browser Pannello SGI sul tuo computer locale.

REQUISITI SISTEMA:
- Windows 10 (versione 1903 o successiva) / Windows 11
- Architettura: x64 o ARM64
- Privilegi amministratore per l'installazione

FUNZIONALITÀ:
- Apertura sicura di documenti PDF, Word, Excel dal browser
- Servizio Windows automatico
- Configurazione guidata durante l'installazione
- Supporto per cartelle documenti ISO personalizzate

SICUREZZA:
- Comunicazione crittografata con il browser
- Nessun accesso internet non autorizzato
- Validazione dei file prima dell'apertura
- Log delle attività per audit
